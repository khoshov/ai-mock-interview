# ai-mock-interview

AI Mock Interview — это интерактивный помощник на базе LLM (Large Language Model) с поддержкой синтеза и распознавания речи, созданный для подготовки к техническим интервью. Пользователь проходит имитацию собеседования в реальном времени: AI задаёт вопросы голосом, воспринимает устные ответы и анализирует их, предоставляя обратную связь.

Особенности:

📢 Голосовое взаимодействие — вопросы и ответы озвучиваются в реальном времени.

🤖 Поддержка LLM (GPT-like) для генерации релевантных технических вопросов.

🎯 Тематические интервью по направлениям (Python, Backend, System Design и др.)

📝 Анализ и оценка ответов с развёрнутой обратной связью.

📊 Финальный отчёт с баллами и рекомендациями.

Идеальный инструмент для тех, кто готовится к интервью в формате one-on-one с реальным техническим собеседником.
