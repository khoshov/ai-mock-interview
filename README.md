## 🎯 Описание

**AI Mock Interview** — это интерактивный помощник на базе LLM (Large Language Model) с поддержкой синтеза и распознавания речи, созданный для подготовки к техническим интервью. 

Система позволяет пользователям проходить имитацию собеседования в реальном времени: AI задаёт вопросы голосом, воспринимает устные ответы и анализирует их, предоставляя детальную обратную связь.

## 🚀 Установка

### Предварительные требования

- Docker и Docker Compose
- Python 3.12+
- Git

### Быстрый старт

1. **Клонировать репозиторий**
   ```bash
   git clone https://github.com/yourusername/ai-mock-interview.git
   cd ai-mock-interview
   ```

2. **Настроить переменные окружения**
   ```bash
   cp .env.example .env
   # Отредактировать .env файл под ваши настройки
   ```

3. **Собрать и запустить контейнеры**
   ```bash
   make build
   make up
   ```

4. **Выполнить миграции**
   ```bash
   make migrate
   ```

5. **Создать суперпользователя**
   ```bash
   make createsuperuser
   ```

6. **Открыть в браузере**
   ```
   http://localhost:8000/admin/
   ```

## 📖 API документация

### Модели данных

#### Category (Категория)
- `name` — название категории
- `description` — описание
- `parent` — родительская категория (иерархия)

#### Question (Вопрос)
- `text` — текст вопроса
- `category` — категория
- `difficulty` — уровень сложности (junior/middle/senior)
- `question_type` — тип вопроса (open/choice)
- `correct_answer` — правильный ответ
- `explanation` — объяснение ответа

### Админ-панель

Доступ к админ-панели: `http://localhost:8000/admin/`

**Возможности:**
- Управление категориями в древовидном виде
- Добавление и редактирование вопросов
- Фильтрация по категориям, сложности и типу
- Поиск по содержимому вопросов
