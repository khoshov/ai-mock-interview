# Тестирование TTS в браузере

## Шаги тестирования:

1. **Откройте http://localhost:8000/interview/**
2. **Откройте Developer Tools (F12) -> Console**
3. **Включите TTS чекбокс** "🔊 Озвучивать ответы интервьюера"
4. **Напишите "старт"** и отправьте

## Что проверить в консоли:

### Ожидаемые сообщения:
```
Отправляем сообщение с TTS: true
Получен аудио чанк, размер: XXXX
Воспроизводим аудио, размер base64: XXXX
Аудио запущено успешно
```

### Если есть ошибки:
- `Ошибка запуска аудио` - проблема с браузером
- `WebSocket error` - проблема с подключением
- Нет сообщений об аудио - TTS не работает

## Тестовые фразы:
- "старт" - начало интервью
- "расскажи о себе"
- "какие у тебя сильные стороны?"
- "опиши свой опыт работы"
